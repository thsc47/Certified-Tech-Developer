## Exercício Aula 2 

Seguem os comando requisitados para a execução da árvore proposta:

- mkdir root
- cd root
- mkdir dir_1 dir_2 dir_3
- cd dir_1
- touch ex01.txt ex02.txt ex03.txt
- mv ex03.txt ../dir_2
- rm ex01.txt
- cd ..
- cd dir_3
- touch ex01.txt ex02.txt ex03.txt
- mv ex03.txt ../dir_2
- rm ex01.txt
- rm -rf root
